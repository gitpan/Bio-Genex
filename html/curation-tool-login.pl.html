NAME

curation-tool-login.pl: a CGI script for downloading a Gene Expression Markup Language (GEML) control bundle file to the curation tool.

SYNOPSIS

  curation-tool-login.pl

DESCRIPTION

This CGI script takes no CGI parameters. It provides a login field and a password and a multi-select list for choosing species for download. It calls control-bundle.pl when the button is clicked.

BUGS

It will not provide a graceful error if the user and password field do not match.

Please send bug reports to genex@ncgr.org
